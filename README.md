# Cannon Game Deep Learning Project

This project explores the use of deep learning techniques in an interactive Cannon Game environment.  
The goal is to apply neural network concepts to game-related decision making and improve the behavior of the system through AI-based methods.

## Technologies Used
- Python
- Deep Learning
- Neural Networks
- Jupyter Notebook

## Project Overview
The project focuses on implementing AI techniques inside a game environment, combining programming, data processing, and neural network concepts.
